# Prefabs for Firebase Cloud Function

These prefabs are meant to be used during with 
Firebase Cloud Functions. They rely on following 
directory structure:

```
.
├── docs
├── docs-output
│   └── placeholder.html
├── firebase.json
├── functions
│   ├── package.json
│   ├── src
│   │   ├── api.ts
│   │   └── index.ts
│   ├── tsconfig.json
│   └── tslint.json
├── README.md
├── secrets.yaml
├── .gitlab-ci.yml
├── .firebaserc
```
